## 📋 Overview

Personal portfolio website showcasing 5 production-grade projects with detailed case studies. Built with a focus on performance, accessibility, and bilingual support (Arabic / English).

---

## ✨ Features

- 🌐 **Bilingual** — Full Arabic (RTL) and English (LTR) support via i18next
- 🌙 **Dark / Light Mode** — System-aware theme switching
- 📱 **Fully Responsive** — Mobile, tablet, and desktop optimized
- 🎬 **Animations** — Smooth transitions with Framer Motion and Lenis
- 📂 **5 Case Studies** — Deep-dive pages for each featured project
- 🔍 **SEO Optimized** — Meta tags, Open Graph, JSON-LD, and sitemap
- ⚡ **Performance** — Lazy loading, optimized assets, clean bundle

## 📁 Project Structure

```
src/
├── assets/
│   └── projects/          # Project screenshots per case study
├── components/
│   ├── sections/          # Page sections (Hero, About, Projects...)
│   ├── layout/            # Navbar, Footer
│   └── ui/                # Reusable UI components
├── data/
│   ├── caseStudiesData.ts  # All case study content (EN + AR)
│   ├── portfolioData.ts    # Main portfolio data
│   └── projectImages.ts    # Project screenshot mappings
├── locales/
│   ├── en.json            # English translations
│   └── ar.json            # Arabic translations
├── hooks/                 # Custom React hooks
└── App.tsx                # Routes configuration
```

## 🏃 Getting Started

```bash
# Clone the repository
git clone https://github.com/ibrahimfares511/portfolio.git

# Navigate to project
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
